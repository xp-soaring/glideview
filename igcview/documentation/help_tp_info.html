Clicking on a turnpoint on the map will pop-up the following window:

Each field on the example window is explained below.

By clicking in the box containing each field, you can edit the name, lat, long of the turnpoint

NAS The abbreviated name for the turnpoint, in the UK called a 'trigraph'. This is the name you usually see displayed on your GPS.
Naseby The longer description of the turnpoint.
52 12.327N The latitude of the turnpoint.
001 39.188W The longitude of the turnpoint.
