If you are a first-time user, there is a step-by-step guide "IGCview for Dummies".

Also there is a guide to install IGCview locally on your PC in the installation notes. If you only access IGCview from home (or work) with a high-speed internet connection, there is no need to download the software and install it, you can run it right from the website.

Here are some notes to help you use IGCview.

To use IGCview, the buttons across the top of the map panel (the green box) in general cause another set of buttons to appear down the left-hand side of the page. There is text associated with each button so if you hover the mouse over the button for a while an explanation of its function will appear. Wherever you see you can click for help related to the current menu.

The top buttons currently are:
File. This gives you options to load flight logs from the web or from your local PC. If you have just launched IGCview from a web site, loading .igc files from the web using the button should be straightforward. If you want to load a file from your local PC you will have to read the installation notes. As IGCview can load .igc or gardown tracklog files, there is also a save file option, which saves the current primary log in the .igc format, including the current task. This is a convenient way to convert your gardown (from Garmin GPS) files to .igc, or to embed a task into an existing .igc file. The rest of the help info for the files menu is contained in the files help file.
Define Task. A menu with various task-related options will appear at the left of the page. After then clicking the 'define task' button, you can define a task by dragging the mouse around the map. You can adjust the task at any time by simply dragging the corners of the task (i.e. turnpoints) or leg mid-points around the map. The rest of the help info for the task menu is contained in the task help file.
Replay. Allows you to maggot-race the loaded logs. To start the replay running, select one of the top two buttons on the left-hand side for either a real-time start or synchronized start (all logs are synchronized to a common start time). Then you can pause, stop, speed up the race with the other buttons. Sometimes logs have a long preamble where the pilot started his logger early in the day. In this case it is a good idea to speed up the replay by hitting the forward button a few times (you will see the clock speed up) and then pause to reset the replay speed when the glider finally gets going. The rest of the help info for the replay menu is contained in the replay help file.
Tools. The button presents a variety of generally useful buttons down the left-hand side. The rest of the help info for the tools menu is contained in the tools help file.
Flight Data. This button will open up a page with a quantative analysis of the logs loaded. Note that IGCview distinguishes between the one primary and (possible multiple) secondary logs. IGCview will compare the primary flight performance with the average of the secondaries. If you only load two flights, then you get a straightforward comparison between the two. If you have many flights loaded, you can use the 'select tracks' button to choose between them, e.g. to select a new primary, and the secondaries for comparison.
Help. Brings up this help page.

To use the map screen, the basic mouse and keyboard actions are:

Click on map IGCview will highlight the nearest log point on the primary trace, and the cursor details will appear down the left of the page.
Space, Backspace keys IGCview will step forwards, backwards through the current primary log.
Click and drag on the map IGCview will either adjust the current task (if one is defined and you clicked near a task turnpoint or leg midpoint), or will draw a zoom box and zoom in when you release the mouse.
arrow keys (up, down, left, right) IGCview will pan the map N, S, W, E by about half a screen
Page Up, Page Down keys Zoom Out, Zoom In
